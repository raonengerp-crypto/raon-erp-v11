# RAON ERP v13.1.1 통합검수 체크리스트

## 배포 전 확인
- ZIP 압축 해제 후 루트에 `package.json`, `server.js`, `public/`가 바로 있어야 함.
- GitHub에는 ZIP 파일 자체가 아니라 압축 해제된 내부 파일을 업로드해야 함.
- v13에서는 `public/app.js`, `public/app.js.tmp`를 사용하지 않음. 남아 있으면 삭제 또는 백업 폴더 이동.

## 견적관리
- 싱글형 견적서 선택 후 용도/구분/형식/모델명이 정상 표시되는지 확인.
- 멀티형 견적서 선택 후 가정용/상업용, 실내기/실외기 모델 목록이 정상 표시되는지 확인.
- 모델 수 배지가 0개가 아닌지 확인.
- 장비 추가 시 품목/제품명/수량/단가/금액이 분리 표시되는지 확인.
- 멀티형 DC율은 내부 화면에서만 보이고 고객 견적 출력에는 표시하지 않는 기준 유지.
- 금액은 천원단위 절삭.

## 현장관리
- 현장 추가/수정/삭제가 되는지 확인.
- 대시보드 KPI와 일정에 반영되는지 확인.

## 자재관리/세무관리
- 자재 DB 표시, 붙여넣기 분류 확인.
- 세무관리 팝업 등록 및 VAT 카드 확인.
